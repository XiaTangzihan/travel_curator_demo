# Day 2 (2024:12:02)

## Event 2 · 杭州西溪喜来登度假大酒店
- sequence: 2
- poi: 杭州西溪喜来登度假大酒店
- short_name: 杭州西溪喜来登度假大酒店
- 类目: 住宿 / 酒店宾馆 / 豪华型酒店
- 文案: 西溪湿地里面的一家万豪，占地面积灰常大，离最近的东岳地铁站有近1km左右。房间是在五楼，挑高大概有5-6m，躺着看房梁的感觉很惬意。有个小阳台，只不过对着的不是山水，可以自行选择关闭门窗。行政酒廊虽然22:30关，但去晚了基本没啥东西了，早餐也得早点去，去晚了想吃特色烧饼的得排大队呀。

# Day 3 (2025:01:05)

## Event 3 · 魁元万通里小炉子地摊烧烤
- sequence: 3
- poi: 魁元万通里小炉子地摊烧烤·东北大油边(余杭万通店)
- short_name: 魁元万通里小炉子地摊烧烤
- 类目: 美食 / 热门/特色菜 / 烧烤
- 文案: 万通底层的一家东北烧烤店，可以选择自己烤或者后厨加工，这个套餐荤素不限共40串，服务员也会友好提醒数量。肉质都很新鲜，火烧得也很快，不一会就烤熟了，注意勤翻面，比如面包片翻完就糊黑了。蘸料有三种，再点个淄博饼子+烤肉，真是绝了!

# Day 7 (2025:07:09)

## Event 7 · 東大方
- sequence: 7
- poi: 東大方(杭州八方城店)
- short_name: 東大方
- 类目: 美食 / 地方菜 / 本帮江浙菜
- 文案: 东大方，名不虚传的一家连锁杭帮菜店，今天有幸来八方城分店体验体验。服务员非常热情，主动引导就座，扫码点单后上菜也很快，如果对菜品有疑问也可以再找服务员咨询。强推红烧肉、香芋煲和红烧沼虾，咸甜鲜得当，很适合下饭，尤其是五常大米！另外提供的茶水也还不错，整体体验👍
